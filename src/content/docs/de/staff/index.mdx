## Rollen

OpenReception kennt bisher nur wenige Rollen. Global Admins sind für die technische Verwaltung, Mandanten-Admins für die Selbstverwaltung innerhalb eines Mandanten und Mitarbeiter:innen für die tägliche Arbeit mit Terminen und Verfügbarkeiten zuständig. Hier ist eine Übersicht, was die einzelnen Rollen tun können:

| Rolle / Berechtigung              | Global Admin | Mandanten Admin | Mitarbeiter:in |
| --------------------------------- | ------------ | --------------- | -------------- |
| Mandanten verwalten               | ✔︎            |                 |                |
| Mandanten-Einstellungen verwalten | ✔︎            | ✔︎               |                |
| Akteure:innen verwalten           | ✔︎            | ✔︎               |                |
| Abwesenheiten verwalten           | ✔︎            | ✔︎               | ✔︎              |
| Kanäle verwalten                  | ✔︎            | ✔︎               |                |
| Mitarbeiter:innen verwalten       | ✔︎            | ✔︎               |                |
| Zugriff auf Termine gewähren      |              | ✔︎               | ✔︎              |
| Termine entschlüsseln             |              | ✔︎               | ✔︎              |
